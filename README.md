# Corona comparison dashboard with plotly and dash

## Requirements

1. poetry >= 0.12
2. pyenv >= 1.2.13
3. python >= 3.6

## Setup

1. `make setup` with poetry for accessing jupyter notebook (creates the .venv environment)
2. `make set` to try the dash app (creates the env environment)
